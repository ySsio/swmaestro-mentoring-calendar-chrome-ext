# 소마 달력 뷰어 크롬 익스텐션

AI·SW마에스트로 접수내역 페이지의 강의 일정을 달력 형식으로 보여주는 크롬 익스텐션입니다.

## 설치 방법

1. 이 폴더(swmaestro-calendar-ext)를 원하는 위치에 저장합니다.
2. 크롬 주소창에 `chrome://extensions` 를 입력합니다.
3. 우측 상단의 **"개발자 모드"** 토글을 켭니다.
4. **"압축해제된 확장 프로그램을 로드합니다"** 버튼을 클릭합니다.
5. 이 폴더를 선택합니다.

## 사용 방법

1. `swmaestro.ai` 접수내역 페이지로 이동합니다.  
   (마이페이지 → 멘토링/특강 게시판 → 접수내역)
2. 화면 우측 하단에 **파란 달력 버튼**이 나타납니다.
3. 버튼을 클릭하면 달력 팝업이 열립니다.
4. 날짜를 클릭하면 해당 날의 강의 목록과 시간이 표시됩니다.
5. 강의 제목을 클릭하면 상세 페이지로 이동합니다.
6. 새로 강의를 접수하거나 취소한 경우 새로 고침 버튼을 눌러야 반영됩니다.
